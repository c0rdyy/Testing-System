int array[];
int matrix[][];
Используя квадратные скобки `[]`
matrix[row][column]
sizeof(array) / sizeof(int)
sizeof(matrix) / sizeof(matrix[0])
`[]`
Тип данных элементов массива, например, `int`, `float`, `char`, и т.д.
Передача указателя на первый элемент массива
Нет, размер массива фиксирован после его объявления.
Одномерный массив имеет одно измерение, а двумерный массив имеет два измерения.
